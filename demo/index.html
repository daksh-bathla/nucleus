Everything You Need

Seven modules. Zero bloat. Just drop in the CSS and start building.

Grid System

12-column CSS Grid with 5 responsive breakpoints and fluid containers.

Typography

Font sizes, weights, alignment, transforms, and line-height utilities.
🔘

Buttons

Solid, outline, ghost variants across 8 colors and 3 sizes.

Cards

Flexible card components with header, body, and footer slots.
📝

Forms

Styled inputs, selects, textareas, checkboxes, and validation states.

Components

Alerts, badges, tables, lists, avatars — everything pre-styled.

Utilities

Spacing, flexbox, display, borders, shadows, opacity, and more.

Dark Mode

Automatic dark mode via prefers-color-scheme — no extra work needed.

Quick Start

One line. That's it.

<link rel="stylesheet" href="nucleus.css">
Then build your layout:
<div class="n-container"> <div class="n-row"> <div class="n-col-4">Sidebar</div> <div class="n-col-8">Main Content</div> </div> </div>
